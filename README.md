## Current Research

**Low-Noise Amplifier for Neural Recording**  
Designed and simulated a neural recording LNA using an NMOS CVF OTA in UMC
180 nm CMOS for implantable BMI applications. The work targets low input-referred
noise and low-power acquisition and has been accepted at the NEleX 2026
International Conference on Next Generation Electronics.

**Optical Fiber Fault Detection**  
Developed a lightweight ANN workflow for automated fault classification from
OTDR traces across eight event categories, structured for near-real-time edge
inference.
